# Blood Suite - Smart Blood Bank Management System

## Overview
Blood Suite is a comprehensive blood bank and donor management system designed for Lesotho. This is a Final Year Project that aims to modernize blood bank operations through technology.

## Project Structure
```
/
├── server.js          # Main Express server (port 5000)
├── public/            # Static frontend files
│   ├── index.html     # Main landing page
│   └── styles.css     # Styling
├── frontend-web/      # Frontend web module (future development)
├── backend/           # Backend API module (future development)
├── database/          # Database schema and documentation
├── mobile-app/        # Mobile application module (future development)
├── ai-ml/             # AI/ML forecasting module (future development)
└── docs/              # Project documentation
```

## Tech Stack
- **Runtime**: Node.js 20
- **Framework**: Express.js
- **Frontend**: HTML/CSS (static files)
- **Port**: 5000

## Running the Application
```bash
npm start
```

## Database Tables (Planned)
1. users - Authentication and base user information
2. donors - Donor-specific information and profiles
3. hospitals - Hospital information and locations
4. blood_inventory - Blood unit tracking and expiry management
5. requests - Blood request management
6. donations - Donation history and records
7. notifications - Alert and notification system
